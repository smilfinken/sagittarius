Projektet
*********
Ett standard Play projekt som har satts upp för NetBeans via kommando "play netbeansify sagittarius".

Vet inte om vi vill köra NetBeans, men det får vi snacka om. Om vi vill köra olika IDEer så får vi göra om projektstrukturen till en standardvariant igen...


Att köra
*********
Körs enklast genom att placera hela Sagittarius-mappen under play-1.2.4-katalogen och köra "play run Sagittarius" -> http://localhost:9000

Att köra test
*************
Se ovanstående, sedan "play test Sagittarius" -> http://localhost:9000/@tests
